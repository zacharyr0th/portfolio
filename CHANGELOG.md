# Changelog

All notable changes to this project are documented in the `changelog` directory, organized by date.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## Recent Changes

See [2025-02-06](changelog/2025/02/2025-02-06.md) for the latest changes.

## Directory Structure

```
changelog/
├── YYYY/               # Year
│   ├── MM/            # Month
│   │   ├── YYYY-MM-DD.md  # Daily changelog entries
```

## Categories

Changes are categorized into:
- Security
- Added
- Changed
- Fixed
- Performance
- Development
